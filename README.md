# Sistema de Gestão de Eventos

## Descrição
Um sistema para criar, gerenciar e visualizar eventos, com funcionalidades para registro de participantes e envio de notificações.

## Tecnologias Utilizadas
- Django
- PostgreSQL
- HTML5
- CSS3
- JavaScript
- jQuery
- Bootstrap
